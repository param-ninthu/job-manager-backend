# Job manager-backend

This is the backend for the job manager. The backend was developed using the SpringBoot along with the MongoDB.
To run this application it's better to use suitable IDE. Eg Intellij,Eclipse Netbeans etc.

## Requirements
* **Maven**
* **Java 17**
* **MongoDB Compass**
* **Clone [Job Manager Frontend]("https://github.com/param-ninthu/job-manager-frontend")**


## Dependencies
There are a number of third-party dependencies used in the project. Browse the Maven pom.xml file for details of libraries and versions used.

## Building the project
* Clone the project from the github
* Install the dependencies
* Connect the application with MongoDB
* Compile the project
* Run the application
